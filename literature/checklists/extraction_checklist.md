# Checklist: Extraction completeness

Confirm you captured:
- Full citation (title/authors/year/venue)
- Problem statement
- Method (core idea + key components)
- Data/setting
- Evaluation metrics
- Findings (only if supported by text)
- Limitations (authors + your inferred)

Red flags
- Any numeric claim without a quote or table reference
- Venue/year guessed from memory
