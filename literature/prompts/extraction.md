# Prompt: Data extraction (per paper)

## Inputs
- Paper PDF or full text (or reliable metadata)
- Extraction schema (fields list)

## Output (fill all fields; use "unknown" if missing)
- Citation: title, authors, year, venue
- Problem statement
- Method
- Data/setting
- Evaluation metrics
- Key findings (quote only if you have the text)
- Limitations stated by authors
- Limitations inferred (clearly labeled)

Rules
- Do not invent numbers, datasets, or results.
- Keep extracted facts separate from your interpretation.
